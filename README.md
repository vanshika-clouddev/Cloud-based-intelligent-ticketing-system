# Cloud-Based Intelligent Ticketing System

 Project Status: Currently Under Development

## Overview
Cloud-hosted support ticket management platform for tracking incidents, managing priorities, and sending automated alerts.

## Tech Stack
- Python
- Streamlit
- AWS EC2
- AWS RDS
- AWS SNS
- MySQL
- GitHub

## Features
- Ticket Creation
- Ticket Tracking
- Priority Management
- Dashboard
- Email Alerts
- Cloud Deployment

## Architecture

User
↓
Streamlit UI
↓
Python Backend
↓
MySQL / AWS RDS
↓
AWS SNS Alerts
↓
Admin Dashboard

## Learning Objectives

- Build a cloud-hosted ticket management platform
- Implement incident tracking workflows
- Integrate AWS cloud services
- Practice cloud deployment and monitoring
- Develop real-world support automation solutions
